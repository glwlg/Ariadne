# Ariadne

Ariadne is the Wails 3 + Vue 3 desktop app that formally replaces the original
Python/PyQt x-tools runtime in this repository.

Before repeating any environment discovery or build work, read section `0.0 开工状态卡`
in `docs/plans/2026-06-13-ariadne-implementation-ledger.md`. It records the
portable Go/Wails paths, current `bin/ariadne.exe` metadata, completed checks,
and the next migration targets.

## Contract Rules

- Search results carry explicit `actions`; the frontend must not infer behavior from `path`.
- Empty launcher queries must stay collapsed and return no seed results.
- Tool centers are launched from search results such as `settings`, `jsondiff`, `hosts`, `net`, `clip ...`, and `wf ...` into independent tool windows; do not add fixed top-level tool buttons back to the launcher and do not route tool centers back into the main launcher window.
- File results may expose `open`, `open_parent`, and `copy_path`.
- Copy-only, QR, calculator, clipboard, and work-memory results must not inherit file actions.
- Lightweight action feedback stays inline in the launcher surface.
